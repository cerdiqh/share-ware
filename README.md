# Share-Wear

A donation platform (client + server) for sharing clothing and items.

## Local setup

1. Install dependencies for root packages (client & server) if needed.

Client:
```
cd client
npm install
npm run dev
```

Server:
```
cd server
npm install
node server.js
```

## GitHub

1. Create a new repository on GitHub (via web or `gh repo create`).
2. Add the remote:
```
git remote add origin https://github.com/<your-username>/<repo>.git
```
3. Push the `main` branch:
```
git branch -M main
git push -u origin main
```
